# WeMovies

## Launch app

Clone the project from GitHub.

From your terminal, execute this commands

```
composer install
npm install
npm run dev
docker-compose up -d
```

Then go to :
- [ ] [http://localhost:8080](http://localhost:8080)



## Stack

- PHP 8.3
- nginx
- bootstrap
- sass / scss
- npm
- webpack
- composer
- docker
